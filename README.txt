スウィーティ ライブアーカイブ サイト一式

■ ファイル構成(すべてリポジトリ直下、flyersだけサブフォルダ)
  index.html         アーカイブ本体(ライブ映像)
  videos.js          ライブ動画データ
  admin.html         ライブ入力フォーム

  shows.html         ライブ予定ページ
  shows.js           予定データ
  shows-admin.html   予定入力フォーム

  songs.html         楽曲リスト
  songs.js           楽曲データ
  songs-admin.html   楽曲入力フォーム

  icon.svg / icon-180.png   サイトアイコン
  robots.txt         admin系ページを検索除外
  flyers/            告知画像フォルダ

■ 公開URL
  https://eitoxuxuxun50.github.io/sweetie-live-archive/

■ 入力フォーム(パスワード制)
  /admin.html         ライブ映像
  /shows-admin.html   ライブ予定
  /songs-admin.html   楽曲
  ※3つとも同じパスワード・同じGitHubトークン設定を共有

■ 更新方法
  各admin.htmlで入力 → GitHubに保存 で自動反映。
  トークンが切れたら「トークン再発行マニュアル.txt」を参照。
